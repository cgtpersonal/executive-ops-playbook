## Operating Philosophy

> An executive should never have to think about logistics. That's my job.

- **Two-week calendar horizon** — meetings and blocks are set before the week begins, not during it.
- **1 big task + 3 small tasks per day** — realistic prioritization over optimistic lists.
- **Every bottleneck is documented** — nothing gets lost, everything gets escalated with context.
- **AI-assisted, human-accountable** — tools handle repetition; judgment stays with the operator.
